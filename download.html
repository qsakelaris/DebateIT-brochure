Download

DebateIt is a web application: you use it in the browser. The course instructions say to skip this section when the product is web-driven—there is no Windows installer or Android APK to distribute here.

Use the live site instead: Open DebateIt (update this link to match your deployment). Source code remains in a private repository; only the hosted app is linked publicly.

Optional: running locally (developers only)

Teammates and graders who need to run the stack locally should follow the README inside your private repo (Django backend, React front end, database, and optional ML model preload). That process is not an end-user download and is not required for the web-presence assignment if the deployed URL is available.
